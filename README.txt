== Description ==

- With the Help of this plugin, you can get post list of all custom post types along with default blog post type posts on your site.
- You can get post list in two format, ListView and GridWiew.
- There will be several more features like ordering posts, selected posts display, pagination etc.
- You can modify default design by adding your own class in "Advanced" section.


== Installation ==

1. Upload `UB Ultimate Post List` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select Block Named "Ultimate Post List" to get list of posts your desired post list.
